## How to run
### VS Code version
DO NOT run main.py when the current directory is SGAI-Outbreak.
You must open the folder SGAI_MK3 with vscode. Then, you can
run main.py from VS Code.
### cmd line version
First, `cd ./SGAI_MK3`. Then, `python main.py`
